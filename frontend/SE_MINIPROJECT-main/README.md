# UrbanWatch — Traffic & Air Quality Dashboard

**Software Engineering Mini Project | Team 9**

## Overview

UrbanWatch is a real-time traffic congestion and air quality monitoring system for Indian smart cities. It provides live data visualization, zone-based analytics, and automated alerts for urban infrastructure management.

## Features

- **Real-time Traffic Monitoring** — Live congestion tracking across city zones
- **Air Quality Index (AQI)** — Live AQI monitoring with health category labels
- **Smart Alerts** — Automated bell-notification alerts for high traffic and poor air quality
- **Zone-based Monitoring** — Filter and monitor specific zones within a city
- **Interactive Maps** — Leaflet-powered dark-theme maps with zone markers
- **Live Charts** — Recharts area/line graphs for traffic and AQI trends
- **Multi-city Support** — 8 major Indian cities pre-configured
- **Geolocation** — Auto-detects and displays user location

## Getting Started

```sh
# Clone the repository
git clone https://github.com/CoderShibu/SE_MINIPROJECT.git

# Navigate to the project directory
cd SE_MINIPROJECT

# Install dependencies
npm install

# Start the development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) and log in with `team9 / team9`.

## Build for Production

```sh
npm run build
```

The output will be in the `dist/` folder, ready to deploy on Vercel, Netlify, or GitHub Pages.
